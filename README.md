## 🎯 **¿Qué es Prestige Beauty Studio?**

Una **aplicación web empresarial completa** desarrollada específicamente para centros de estética y salones de belleza. Integra todas las operaciones comerciales en una plataforma unificada que optimiza la gestión diaria y maximiza la rentabilidad del negocio.

### 🏆 **Características Destacadas**
- ✅ **Sistema de autenticación** con roles diferenciados (Admin/Cliente)
- ✅ **Gestión completa de inventario** con control de stock en tiempo real
- ✅ **Dashboard de estadísticas profesionales** con métricas avanzadas
- ✅ **Sistema de ventas integrado** con múltiples métodos de pago
- ✅ **Gestión de turnos y citas** con calendario interactivo
- ✅ **Interfaz elegante** con diseño premium dorado y gris
